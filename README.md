# 🐍 Flask + MySQL Docker Application

A Dockerized Flask web application integrated with MySQL using Docker Compose. This project was developed as part of a Cloud Computing coursework assignment to demonstrate multi-container deployment, database connectivity, and container orchestration.

---

## 📖 Project Overview

This project demonstrates how to deploy a Flask web application alongside a MySQL database using Docker Compose. The application communicates with the database through a Docker network and uses persistent storage for database data.

## 🚀 Running the Project

### Build

```bash
docker-compose build
```

### Start

```bash
docker-compose up
```

Application

```
http://localhost:5000
```

---

## Stop

```bash
docker-compose down
```
